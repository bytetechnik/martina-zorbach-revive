# Martina Zorbach Coaching Website

Professional coaching website for Martina Zorbach - Diplom-Psychologin & Systemische Coachin in Frankfurt am Main.

## Project Overview

This is a modern, responsive website built for Martina Zorbach's coaching practice, offering services for leadership coaching, career coaching, and life transitions.

**Website**: https://martinazorbachcoaching.de/

## Technologies

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - High-quality component library
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn package manager

### Installation

1. Clone the repository:
```sh
git clone <YOUR_GIT_URL>
```

2. Navigate to the project directory:
```sh
cd "Martina Life Coach"
```

3. Install dependencies:
```sh
npm install
```

4. Start the development server:
```sh
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run build:dev` - Build in development mode
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint to check code quality
